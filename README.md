# Simple Calculator

A lightweight command-line calculator that performs basic arithmetic operations.

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modular division (%)
- Clear previous result
- Exit functionality

## Requirements
- Python 3.x
